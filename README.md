##  Using Assets in Skool

### Direct Link Format
When hosting on GitHub or similar:
```
https://raw.githubusercontent.com/[username]/AI-Video-Factory-Resources/main/[path-to-file]
```

### Embedding in Skool Lessons
1. Upload video to GitHub or hosting service
2. Copy raw file URL
3. Paste into Skool lesson editor
4. Use markdown or HTML embed format

##  File Naming Conventions

- **Videos**: `##-descriptive-name.mp4` (e.g., `01-intro-to-advanced-tools.mp4`)
- **Templates**: `descriptive-name-template.md/.pdf` (e.g., `workflow-template.md`)
- **Scripts**: `descriptive-name.sh/.py` (e.g., `batch-export.sh`)
- **Docs**: `descriptive-name.md/.pdf` (e.g., `ethics-whitepaper.pdf`)

##  Quick Start

1. **For Content Creators**: Navigate to your course folder and add assets
2. **For Students**: Access files via direct links in Skool lessons
3. **For Admins**: Update this README when adding new files
