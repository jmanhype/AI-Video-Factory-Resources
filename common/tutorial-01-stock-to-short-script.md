## Production Notes

### Screen Recording Checklist
- [ ] Record in 1920x1080 (or 2560x1440 for Retina)
- [ ] Hide desktop clutter, close unnecessary tabs
- [ ] Use CapCut screen recorder or OBS
- [ ] No audio capture (avatar provides narration)
- [ ] Record full 10-minute build in real-time

### CapCut AI Avatar Setup
- [ ] Use custom avatar (your photo/video) OR stock professional presenter
- [ ] Paste full script (sections 0:00-10:00)
- [ ] Generate avatar video
- [ ] Download as separate MP4

### Final Edit Assembly
- [ ] Import screen recording
- [ ] Import avatar video
- [ ] Picture-in-picture: avatar bottom-right corner (20% screen size)
- [ ] OR split-screen: avatar left 30%, screen recording right 70%
- [ ] Sync avatar timing with screen actions
- [ ] Add intro card: "Stock ‚Üí Short in 10 Minutes" (3 seconds)
- [ ] Add outro card: "Post yours in #proof-wednesday" (3 seconds)

```
Resolution: 1920x1080 (16:9 for YouTube/Facebook)

Frame Rate: 30 FPS

Quality: High
```
