## Platform Export Settings

### TikTok / Instagram Reels / YouTube Shorts
```
Resolution: 1080x1920 (9:16)
Frame Rate: 30 FPS
Format: MP4 (H.264)
Max Duration: 60 seconds (TikTok), 90 seconds (Reels/Shorts)
Max File Size: 50 MB
```

### YouTube (Landscape)
```
Resolution: 1920x1080 (16:9)
Frame Rate: 30 FPS
Format: MP4 (H.264)
```
